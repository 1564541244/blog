div和span标签

<div>标签：div是一个块级元素，这意味着它的内容自动地开始一个新行。标签可以把文档分割为独立的、不同的部分。它可以用作严格的组织工具，并且不使用任何格式与其关联。如果用 id 或 class 来标记<div> ，那么该标签的作用会变得更加有效。

<span>标签：span标签被用来组合文档中的行内元素。
